    Javascript based Main Menu

    The menu for the website is powered by two main JS components.
    The first will first load the HTML code in to a HTML element to build the menu.
    After the page has loaded, the second part of the Javascript menu will run,
    giving the menu its functions.

    Why I chose this method

    Making the header like this has allowed me to maintain
    flexibility in future-proofing the site. I can keep my main menu
    up to date on every page of the site, without the need for PHP.
    The JS that powers the menu itself also remains fully functional.
